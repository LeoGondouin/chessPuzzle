Application permettant de créer et résoudres des problèmes d'échecs.


Mode d'emploi du "Solver" : Simple, résolvez des puzzles, si vous vous trompez vous perdez des points, si vous le résolvez vous en gagnez.
Si vous vous êtes trompés ou vous êtes arrivés à la fin de la séquence, cliquez sur le bouton "Next puzzle" afin de résoudre le prochain problème


Mode d'emploi du "Manager" : Placez d'abord vos pièces par FEN code ou manuellement, une fois que vous êtes prêt à enregistrer la solution, cliquez sur "Start sequence", si la tous les critères sont validés vous pourrez à présent bouger les pièces afin d'indiquer la solution. Une fois que la solution a été renseignée, il vous faut appuyer sur save puzzle afin d'enregistrer le puzzle créé.


Veuillez consulter le fichier "fonctionnalités.xlsx" afin d'avoir l'état des fonctionnalités voulues en détail avec les contraintes associées
Pour consulter le rapport du projet merci de consulter "Rapport de projet.pdf"

Le vrai dossier de projet est dans le dossier "chessPuzzle".
Merci de consulter juste la partie Main sans regarder le code pour le dossier "chessPuzzle-design.zip" afin de consulter la maquette de l'application qui n'a pas pu être adaptée à la version du dossier "chessPuzzle".
